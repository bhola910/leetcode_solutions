# Changelog

All notable changes to this project will be documented in this file.

---

## [0.2.0] - 2026-07-03

### Added

- Professional Logger module.
- Console logging support.
- File logging support.
- Custom log formatting.
- Public logging methods:
  - `info()`
  - `warning()`
  - `error()`
  - `debug()`

---

## [0.1.0] - 2026-07-02

### Added

- Project structure.
- Python virtual environment.
- ConfigManager.
- Configuration loading.
- Configuration validation.
- Nested configuration access (`get()`).
- Configuration reload support.

## [0.3.0] - 2026-07-06

### Added

- GitManager
- Git status support
- Git add support
- Git commit support
- Git push support
- Git pull support
- Current branch detection
- Repository clean check
- Shared Git command executor

## [0.4.0] - 2026-07-07

### Added

- FileWatcher service
- Watchdog integration
- File extension filtering
- Debounce mechanism
- Current branch detection
- Repository clean status check

### Improved

- Refactored GitManager using a shared Git command executor
- Improved logging across modules
